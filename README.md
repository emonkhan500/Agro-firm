# Agro-Firm

Agro-Firm is a modern agriculture-based web application built with **Next.js App Router** and **Server Actions**.  
The platform provides a clean interface where users can explore agricultural products and manage content through an admin dashboard.

## Live Website
https://agro-firm.vercel.app/

##  Tech Stack

- Next.js (App Router)
- React.js
- Tailwind CSS
- JavaScript
- Next.js Server Actions

##  Features

-  Modern agriculture themed UI
-  Fast performance with Next.js
-  Data handling using **Server Actions**
-  Fully responsive design
-  Admin Dashboard for content management
-  Agro products showcase
-  Clean and modular component structure

##  Run Locally

Clone the repository

```bash
git clone https://github.com/emonkhan500/Agro-firm.git
```

Go to the project directory

```bash
cd Agro-firm
```

Install dependencies

```bash
yarn install
```

Run the development server

```bash
yarn dev
```

Build for production

```bash
yarn build
```

Start production server

```bash
yarn start
```
